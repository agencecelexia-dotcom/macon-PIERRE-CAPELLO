## NOTES TECHNIQUES

- **Format** : JPG, qualité 85-90%, optimisé pour le web
- **Résolution recommandée** :
  - 16:9 → 1920×1080px
  - 4:3 → 1600×1200px
  - 3:2 → 1800×1200px
  - 1:1 → 800×800px
  - 21:9 → 2520×1080px
- **Poids cible** : < 300 Ko par image après compression (Next.js optimise aussi via `next/image`)
- **Nommage** : respecter exactement les noms de fichiers ci-dessus, ils correspondent aux chemins dans le code
- **Dossier** : placer dans `public/images/` en respectant l'arborescence (`services/`, `portfolio/`, `blog/`, `team/`, `og/`)
